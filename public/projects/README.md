# Project Images

Add your project screenshots/images here with the following names:

1. `asset-management.jpg` - Screenshot for Asset Management Dashboard
2. `tourist-stay.jpg` - Screenshot for Tourist Stay Finder
3. `chat-app.jpg` - Screenshot for Real-Time Chat Application

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 800x600px or larger
- Aspect ratio: 16:9 or 4:3 works best
- File size: Keep under 500KB for faster loading

## How to add images:
1. Take screenshots of your projects
2. Resize/optimize them if needed
3. Save them in this folder with the names above
4. The portfolio will automatically display them!
